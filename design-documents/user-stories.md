# User Stories

## MVP User Stories

### Site Content
#### About *
- [ ] As a user, I want to see an 'About' page to learn more about the site

#### Tutorial *
- [ ] As a user, I want a page to learn about the game's rules and gameplay


### Account Management and Signup
#### Sign up *
- [x] As a new user, I want to be able to create a new profile to play GroovyBones

#### *Game Stats *
- [ ] As a registered user, I want to be able to read cumulative game records/stats from past games

#### *Game Management *
- [ ] As a registered user, I want to be able to save(add) unfinished games or delete them

#### Sign In *
- [x] As a registered user, I want to be able to sign in to my profile and authenticate it

#### Sign Out *
- [x] As a registered user, I want to be able to sign out of my profile (both session and Cognito sign out)

#### Password Resets *
- [x] As a registered user, I want to be able to reset passwords if they're lost or forgotten


### Gameplay
#### Gameplay *
- [x] As a user, I want to be able to play against a computer-controlled opponent

#### Difficulty *
- [x] As a user, I want to play against an 'intelligent' computer-controlled opponent


## NON-MVP User Stories
### Gameplay
#### Difficulty
- [x] As a user, I want to have options for different difficulty levels

#### Multiplayer
- [ ] As a user, I want to be able to play against other, real, people



### Game Modifiers
#### Game Board Size
- [ ] As a registered user, I want to be able to increase the board size beyond 3x3

#### Game Dice Range
- [ ] As a registered , I want to be able to adjust the range of dice values present

#### Add Modifier
- [ ] As a registered user, I want to add new game modifiers to add to games

#### Delete Modifiers
- [ ] As a registered user, I want to be able to delete saved modifiers

#### Edit Modifiers
- [ ] As a registered user, I want to be able to edit current modifiers
