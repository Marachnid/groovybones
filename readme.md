## Simple Grails App
Reusable framework for new Grails webapps w/GORM and MySQL

Groovy 3.0.25
Grails 6.2.3 w/embedded Tomcat
Gradle 7.6.4
GORM 8.1.2
MySQL 8.4.6
SLF4J (included automatically)


Simple Repo to practice with Grails apps and GORM
- Generated with Grails create-webapp command
- Removed prior content for the default landing page
- Added a few controllers and basic pages for MVC navigations
- Added an application-secrets file for DB credentials
